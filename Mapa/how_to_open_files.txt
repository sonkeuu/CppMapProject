Obliczyć NWW (najmniejszą wspólną wielokrotność, patrz algorytm poniżej) dwóch liczb naturalnych podanych w pamięci pod adresem symbolicznym A i B.

int NWW (int a, int b) {
int a1 = a, b1 = b;
while (a1 != b1) {
     if (a1 > b1)
            b1 = b1 + b;
     else
            a1 = a1+ a;
     }
return a1;
}
